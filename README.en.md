# Raspberry-Car

#### Description
该小车实现了 自动避障，实时图像传输(网络摄像头)，目标检测（yolo5），ui控制（QT）
硬件采用树莓派4B, L298N驱动，CSI树莓派摄像头，6个红外传感器，4个轮胎+4个马达，12V电源+充电宝
